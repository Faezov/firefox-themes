# Impressionist Water Garden Bulat Theme v0.1.0

Animated impressionist water-garden Firefox theme with soft dawn light, lily pads, floating brushstrokes, and shimmering reflections.

- Original theme art inspired by impressionist color and atmosphere, not copied from any specific painting.
- Uses a 5000x200 animated SVG header image.
- Keeps the left side dark for readable Firefox chrome.
- No scripts, no permissions, no tracking.

Install: `about:debugging#/runtime/this-firefox` -> Load Temporary Add-on -> `manifest.json`
