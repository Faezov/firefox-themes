# Southern Cross Night Bulat Theme v0.1.0

Animated southern night sky Firefox theme with a bright Southern Cross, soft rain, and a dark forest horizon.

- Uses a 5000x200 animated SVG header image.
- Keeps the left side darker for readable Firefox chrome.
- No scripts, no permissions, no tracking.

Install: `about:debugging#/runtime/this-firefox` -> Load Temporary Add-on -> `manifest.json`
