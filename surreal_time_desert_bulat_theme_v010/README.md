# Surreal Time Desert Bulat Theme v0.1.0

Animated surreal night-desert Firefox theme with melting clock shapes, floating stars, and warm dreamlike shadows.

- Uses a 5000x200 animated SVG header image.
- Keeps the left side dark for readable Firefox chrome.
- No scripts, no permissions, no tracking.

Install: `about:debugging#/runtime/this-firefox` -> Load Temporary Add-on -> `manifest.json`
