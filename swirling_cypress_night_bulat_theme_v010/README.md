# Swirling Cypress Night Bulat Theme v0.1.0

Animated post-impressionist Firefox theme with cobalt sky swirls, golden starbursts, cypress silhouettes, and warm wheat-field strokes.

- Original theme art inspired by expressive post-impressionist color and motion, not copied from any specific painting.
- Uses a 5000x200 animated SVG header image.
- Keeps the left side dark for readable Firefox chrome.
- No scripts, no permissions, no tracking.

Install: `about:debugging#/runtime/this-firefox` -> Load Temporary Add-on -> `manifest.json`
