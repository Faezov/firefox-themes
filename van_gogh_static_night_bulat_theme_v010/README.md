# Van Gogh Static Night Bulat Theme v0.1.0

Static post-impressionist Firefox theme with a cobalt night sky, golden starbursts, cypress shadow, and warm wheat-field brushwork.

- Uses a 5000x200 PNG header image.
- Original generated artwork inspired by expressive post-impressionist night landscapes, not copied from any specific painting.
- Keeps the left side dark for readable Firefox chrome.
- No scripts, no permissions, no tracking.

Install: `about:debugging#/runtime/this-firefox` -> Load Temporary Add-on -> `manifest.json`
